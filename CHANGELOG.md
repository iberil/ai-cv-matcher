## [1.0.0] - 2024-01-XX

### Added
- Initial release of CV-Matcher
- Complete CV analysis pipeline with fallback systems
- ATS compliance scoring and recommendations
- Machine learning-based job matching
- FastAPI backend with PostgreSQL
- Next.js frontend with Tailwind CSS
- Docker support for easy deployment
- Comprehensive API documentation
- Interactive career coaching powered by Qwen2.5-72B

### Features
- Job seekers can upload and analyze CVs
- Get ATS compliance scores and recommendations
- Find matching job opportunities
- Track job applications
- Receive AI-powered career advice
- Employers can post jobs
- Employers can find matching candidates
- Competitor analysis and salary insights
